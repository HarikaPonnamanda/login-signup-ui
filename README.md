# Login & Signup UI

Frontend login and signup system with client-side validation and LocalStorage.

Features
- Signup with validation (email format, password match)
- Login using stored credentials in `localStorage`
- Redirects to a demo `home.html` on successful login

Tech
HTML, CSS, JavaScript

How to run
1. Open `signup.html` to create an account.
2. Then open `login.html` and sign in with the same credentials.
